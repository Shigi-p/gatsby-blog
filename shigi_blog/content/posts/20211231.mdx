狂うことは出来なかったのではないかと思います(？)

迷いが多くなんか一つのことに集中して取り組むというよりも、なにかしては飽きてなにかしては飽きてを無限に繰り返していたような気もするので、もうちょっと一つに集中したりまともに過ごせたらいいなぁと思っている。

現状は「最強になる」とかを目標にしておこうかなとか適当なことを考えている。

### Hobby

#### FFⅩⅣ

今年頭から初めて、11月あたりに蒼天クリアしました(遅)

弓術士をメインジョブにしようと思ってたけど、途中から双剣士に変えて忍者でずっと来ていました。

唯一の悔いは「オルシュファン卿！」を適当な会話しながら流し見してしまったことくらいでした。

1月には紅蓮に入りたいとは思いますがどうなることやら

#### Uma musume Pretty derby

アニメ2期見たらめちゃクソハマりました。

おかげでツインターボのシーンと「ユメヲカケル！」で反射で泣く体になってしまいました。

推しはナイスネイチャです。

#### Music

音楽の幅を広げよう広げようと思って結局広がらなかったような年でした。

ラップを聞こう聞こうと思いつつ結局バトルMCから広がらなかったり。

一番印象に残ったのはキタニタツヤの聖者の行進。

「平穏世代の韋駄天達」のOP映像がばちこり電子ドラック感がありつつ、恐ろしい色彩で殴ってくる感じが最高。

### Tech

個人的な面で言えば技術的なことはマジで全然やっていない気がしている

働き始めて業務であれこれ触ったけど、自分ではなんとも言えない成長だと思っている………うごごごご

#### React

業務でhooksを基本使っていることでhooksの記述自体には慣れたのではないかと思う

とかいいつつ、reactの細かい仕様のあれこれを全く理解していないのは完全に課題だと思う。

- useMemo、useCallbackによるレンダリングの削減とか？
- 未だに惰性で書いている部分があるのでそれぞれ理解しながら使用することかなぁ…

#### Typescript

typescriptを完全にノリで使っているフシがある…

何がなんだかわからない状態からちょっと進んで、これをまずは理解するべきだなという課題が浮き彫りになってきた。

- Utility Typeを何一つ理解していないのは大きな問題だと思った
- 型についての理解を深める

#### HTML

何を学んだのかと言われるとそんなに無いんだけど、あまり使用しないようなディープなタグとか要素とかちょこちょこ仕入れたり、正しいDOM構造とはなにかみたいな部分は知れたので成長っちゃあ成長ではある。

#### CSS

個人的に好きだしやってて楽しいんだけど、一番学習するのが複雑な分野。

CSSの設計とか極めたところでなんとも言えない感がある…。

IEの対応とか、自分の中でこうすればいいんじゃないか的な知見が溜まってきたので、これをベースにあれこれ拡張していきたい所存。

- FLOCSとかCSS設計を**参考程度に**見てみる
- あくまでもこれはUIを良くする手段であって、より深めるところは別にあるというスタンスで行ったほうが良いのかも知れない

### Next year

毎年1月はやる気みちみちなので、これを1月中持続できるようには頑張りたい…。

毎日頑張れば全然エネルギー出るんだから、それなりにエネルギー出して生きていこうかと思います。

良いお年を。
