## 事の発端

働き始めてから俄然フロント勉強する気持ちが高まったり、高まってなかったりしてきた。

「久々に Gatsby でも勉強するか」と思い、1 年くらい前に思いつきで Gatsby を触っているリポジトリを見てみる。「あ〜〜〜こんなのあったな〜〜〜」って感じでざーっと見て、触るため clone。以前 Linux を使っていた時のリポジトリ。僕は今年から M1 搭載の Mac を使い始めたので clone して持ってこないと編集できないのである。

clone、npm install。

**なんか通らない。**

M1 Mac 特有のエラーがいくつも出てくるようで、node の version 関係だとか、chrome は m1 対応のものをインストールしなければならないだとか(これは別の場所のエラーだった気もするが)
とにかく色々エラーが多い。

乗り越えて gatsby develop。去年の僕は gatsby の標準そのままイチから blog を作ろうとしていたようでなんか大変そう(他人事)。

というわけでテンプレートを使ってまた作り直すことに。

template を引っ張ってくる。

gatsby new blog ...

**なんか通らない。**

面倒である。gastby new でプロジェクトを作った後の npm install で引っかかっているようで？？？とお手上げ。--force という禁断のオプションを使い事なきを得た。さてはて gatsby develop で確認。
とってもシンプルなテンプレートを持ってきたので特別感動は無いがお手軽感を味わえる。

調べてみると、Gatsby Cloud なる簡単にデプロイできるものがあるというではないか。当初は Github pages でも使うかと考えてたけど、どうやら便利そうだしこちらを試すことに。

公式ページからポチポチ。流れで行けてしまった。特に苦労もなく。というわけでこのブログが出来上がる。

今ここ。

## 何書くか

技術系の詰まったこととかをリポジトリ直下の適当な README.md に残しておく癖があるので、それをこっちに書き綴ろうかと思う。最近は働き始めたのと M1 Mac で詰まるところがたくさんあるので書き放題。

しょーもない技術のことからオタクの語りまで暇つぶしに書く。
